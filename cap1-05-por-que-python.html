🐍 Por que Python?

Seção 1.5 — A linguagem que conquistou o mundo pela simplicidade

✨ Simplicidade e legibilidade

Python foi criada por Guido van Rossum em 1991 com um objetivo claro: ser uma linguagem fácil de ler e escrever. Enquanto outras linguagens usam chaves, ponto-e-vírgula e símbolos complicados, Python usa indentação e palavras que parecem inglês.

🐍 Python if idade >= 18: print("Maior de idade") else: print("Menor de idade")
☕ Java if (idade >= 18) { System.out.println("Maior de idade"); } else { System.out.println("Menor de idade"); }

Percebeu a diferença? O código Python é mais limpo, mais curto e mais próximo do português. Isso não é acidente — é filosofia de design.

🔋 “Batteries Included”

Python vem com uma biblioteca padrão enorme — isso significa que muita coisa já vem pronta para usar, sem precisar instalar nada extra. Os programadores chamam isso de “batteries included” (pilhas inclusas).

🚀 Seu primeiro programa

Em muitas linguagens, o primeiro programa que um iniciante escreve é o famoso “Hello World”. Em Python, é absurdamente simples:

Pythonprint("Olá, Mundo!")

Sim, é só isso. Uma única linha. Execute esse comando e você verá:

SaídaOlá, Mundo!

Agora vamos um pouco além. Que tal um programa que pergunta seu nome?

Python# Meu primeiro programa interativo nome = input("Qual é o seu nome? ") print(f"Prazer em conhecê-lo, {nome}!")
